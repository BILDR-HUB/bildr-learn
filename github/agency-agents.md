# agency-agents

**Kategória:** `agent gyűjtemény` / `AI agency`
**URL:** https://github.com/msitarzewski/agency-agents

---

## Mi ez és mire jó?

> [!tldr] Egy mondatban
> Komplett AI agency csapat - minden szerepre specializált agent (frontend, backend, copywriter, SEO, social media, QA), mindegyik saját személyiséggel, folyamatokkal és deliverable-ökkel.

Az agency-agents egy kész "virtuális ügynökség": nem egy eszköz, hanem **agent definíciók gyűjteménye**. Minden agent egy konkrét szerepet tölt be (frontend wizard, Reddit community ninja, SEO specialist, stb.), és tartalmazza a prompt-ot, a workflow-t, és az elvárt output formátumot.

Ha saját AI agent csapatot építesz, ez inspiráció ahhoz, hogyan strukturálj agent szerepeket, milyen felelősségi körök kellenek egy digitális ügynökségnél.

---

## Mikor hasznos?

- **Agent szerepek tervezése** - ha saját agent team-et építesz (akár [[toolbox/claude-code-agent-teams|Claude Code Agent Teams]]-szel), innen inspirálódhatsz a role definíciókon
- **Kiegészítő feladatok automatizálása** - copywriting, SEO audit, social media tartalom agent-ekkel
- **Agency workflow építés** - ha automatizált szolgáltatásokat akarsz AI agent-ekkel

### Mikor NE használd

- Közvetlenül kódolásra - ezek nem [[toolbox/claude-code|Claude Code]] skill-ek, hanem agent prompt-ok
- Ha egyedi, mély szakértelem kell - az agent-ek generalisták, nem domain expert-ek

---

## AI-natív fejlesztés

Az agent definíciók adaptálhatók [[toolbox/claude-code-agent-teams|Claude Code Agent Teams]] teammate prompt-okként. A szerepkiosztás mintája (frontend / backend / QA / copywriter) átültethető a saját team struktúrádba.

> [!tip] Hogyan használd AI-val
> "Spawn-olj egy teammate-et ezzel a szereppel: [agency-agents repo-ból kimásolt role definition]" - így az agent-ek struktúráját közvetlenül felhasználhatod a saját workflow-dban.
