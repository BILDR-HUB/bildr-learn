# Everything Claude Code

**URL:** https://github.com/affaan-m/everything-claude-code
**Stars:** ~83K

> [!tldr] Egy mondatban
> All-in-one [[toolbox/claude-code-projekt-setup|Claude Code]] optimalizációs rendszer - skills, instincts, memory, security, és research-first fejlesztési módszertan egy repóban.

---

## Mi ez és mire jó?

Nem egy tool, hanem egy **tudásbázis + skill gyűjtemény**: hogyan hozd ki a maximumot a Claude Code-ból. Tartalmaz skill-eket, "instinct"-eket (automatikus viselkedési minták), memory rendszert, security best practice-eket, és research-first fejlesztési módszertant.

---

## Mikor hasznos?

- **Inspiráció** - új skill-ek, workflow-k, és pattern-ek felfedezése
- **Security hardening** - Claude Code session-ök biztonságosabbá tétele
- **Memory rendszer ötletek** - hogyan kezeld a projekt kontextust session-ök között
- **Onboarding** - ha valaki új a Claude Code-ban, ez jó kiindulópont

### Mikor NE használd

- Mint kész, változatlan rendszert - válogass belőle, ne másold teljes egészében
- Ha már van kialakult workflow-d - túl sok változtatás egyszerre kontraproduktív

---

## Claude Code kontextus

Közvetlenül Claude Code-ra épül. A repó skill-jei telepíthetők, de értékesebb **referenciának** használni: nézegesd, amit jónak tartasz azt adaptáld a saját setup-odba.

> [!tip] Hogyan használd
> Nem promptban használod - inkább böngészd a repót, és ami releváns, azt integráld a saját `~/.claude/skills/` vagy CLAUDE.md-be.
